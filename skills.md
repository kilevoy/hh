## 📋 Как использовать на hh.ru

### 1. Поле «Навыки» (до 30 штук)

**Вставьте этот блок (приоритетные сверху):**

```
Управление продажами, Управление командой, CRM, Бюджетирование, 
Стратегическое планирование, Управление проектами, Переговоры, 
B2B продажи, Анализ рынка, Управление персоналом, Мотивация персонала, 
Бизнес-планирование, Управление филиалами, Запуск проектов, 
Внедрение CRM, MS Power BI, 1C, amoCRM, Управление изменениями, 
Антикризисное управление, ISO 9001, Дашборды, Управленческая отчётность, 
Подбор персонала, Обучение персонала, Разработка регламентов, 
Федеральные сети, Масштабирование, Оптимизация процессов, KPI
```

### 2. Поле «Опыт работы» — добавьте навыки в описание

**Пример для «ИНСИ»:**

> **Навыки в работе:** Управление продажами, CRM (1C CRM, amoCRM, MS Dynamics), Бюджетирование, Стратегическое планирование, Управление филиалами, Масштабирование, Антикризисное управление, MS Power BI (дашборды для руководителей)

### 3. Пройдите тесты hh.ru (совет №5)

**Рекомендуемые тесты:**
- [ ] Управление персоналом
- [ ] Управление продажами
- [ ] Переговоры
- [ ] Тайм-менеджмент
- [ ] 1С:Предприятие (если есть)

**Где:** hh.ru → Кабинет соискателя → Мои тесты

## 📋 Как использовать в LinkedIn

### 1. Раздел «Skills» (до 50 навыков)

Добавьте все 50 навыков из списка выше. LinkedIn позволяет до 50.

### 2. Раздел «About»

**Включите ключевые слова естественно:**

> «Руководитель по продажам с 18-летним опытом управления командами до 85 человек и федеральной сетью из 26 филиалов. Экспертиза: стратегическое планирование, бюджетирование, внедрение CRM (1C CRM, amoCRM, MS Dynamics), антикризисное управление, масштабирование бизнеса, цифровизация процессов продаж.»

### 3. Раздел «Experience»

**В описании каждой позиции добавьте навыки:**

> **Ключевые навыки:** Управление продажами • CRM • Бюджетирование • Управление командой • Масштабирование • Антикризисное управление • MS Power BI
