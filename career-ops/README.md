# Карьерный штаб

Отдельное веб-приложение для поиска работы и печатных версий резюме.

## Как открыть

Локально:

```text
career-ops/index.html
```

Через GitHub Pages после публикации репозитория:

```text
https://kilevoy.github.io/hh/career-ops/
```

## Что внутри

- панель задач поиска работы;
- воронка вакансий;
- трекер интервью;
- профиль кандидата;
- два варианта резюме:
  - `Резюме` — классический ATS-friendly формат;
  - `Резюме 2` — вариант в стиле структурированного документа с таблицами.

Оба варианта резюме имеют кнопку `Печать / PDF`.
