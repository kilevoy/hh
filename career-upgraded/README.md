# Career Upgraded

Это усиленная версия карьерных документов.

Что внутри:
- `resume/` — улучшенное резюме и достижения
- `stories/` — STAR-истории в интервью-формате
- `cover-letters/` — шаблоны сопроводительных писем
- `interviews/` — ответы на сложные вопросы
- `companies/` — стратегия по целевым компаниям
- `skills.md` — навыки с доказательной базой
- `linkedin.md` — позиционирование для LinkedIn

Принцип:
- Оригиналы не изменены.
- Здесь лежат версии «готово к отклику».
