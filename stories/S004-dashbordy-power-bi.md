# S004 — Дашборды для руководителей (MS Power BI)

## STAR-формат

### **Situation (Ситуация)**
Руководство компании «ИНСИ» не видело реальную картину продаж в реальном времени. Отчётность готовилась вручную в Excel, данные устаревали к моменту получения, решения принимались с задержкой.

**Проблема:**
- Отчётность в Excel — долго, ошибки при вводе
- Данные устаревают к моменту получения
- Руководство не видит воронку продаж в реальном времени
- Сложно отслеживать KPI менеджеров и филиалов
- Решения принимаются с задержкой

---

### **Task (Задача)**
- Разработать дашборды для руководителей на базе MS Power BI
- Обеспечить визуализацию ключевых метрик в реальном времени
- Автоматизировать сбор данных из CRM и 1С
- Сократить время на подготовку отчётности

---

### **Action (Действия)**

**1. Анализ потребностей:**
- Определил, какие метрики нужны руководству (выручка, воронка, конверсия, KPI)
- Выявил источники данных (CRM, 1С, Excel-отчёты)
- Согласовал формат визуализации с руководством

**2. Разработка дашбордов:**
- Создал дашборды в MS Power BI:
  - **Общая панель:** выручка, план/факт, динамика
  - **Воронка продаж:** конверсия по этапам
  - **Филиалы:** сравнение показателей между филиалами
  - **Менеджеры:** KPI, выполнение плана, активность
  - **Продукты:** продажи по категориям, маржинальность

**3. Интеграция данных:**
- Настроил подключение к 1C CRM
- Автоматизировал обновление данных (ежедневно)
- Обеспечил доступ с любых устройств (облако)

**4. Внедрение:**
- Обучил руководство работе с дашбордами
- Настроил рассылку отчётов по расписанию
- Продолжаю поддержку и развитие

---

### **Result (Результат)**
- **Прозрачность:** Руководство видит все метрики в реальном времени
- **Скорость решений:** Данные актуальны на текущий момент, а не на неделю назад
- **Автоматизация:** Отчётность обновляется автоматически, без ручного ввода
- **KPI под контролем:** Отклонения видны сразу, можно быстро реагировать
- **Масштабируемость:** Можно добавлять новые метрики и дашборды

---

### **Earned Secret (Инсайт)**

> «Дашборд не для красоты — он для решений. Если руководитель смотрит и не понимает, что делать — дашборд не работает.»

> «Автоматизация отчётности — это не про экономию времени. Это про то, чтобы данные были актуальными в момент принятия решения.»

> «Power BI выигрывает у Excel там, где нужна скорость обновления и визуализация. Excel — для анализа, Power BI — для мониторинга.»
