# PROJECT STRUCTURE

## 1) Рабочая зона поиска работы

- `resume/`
  - `resume-main.md` — актуальное резюме
  - `achievements.md` — достижения для усиления резюме
- `stories/` — STAR-кейсы
- `cover-letters/` — шаблоны сопроводительных писем
- `interviews/` — подготовка к интервью
- `companies/` — целевые компании
- `linkedin.md` — профиль LinkedIn
- `skills.md` — навыки и формулировки

## 2) Методическая зона Interview Coach

- `SKILL.md` — главный документ навыка
- `references/` — база методик и команд
- `releases/` — релизные заметки
- `VERSIONS.md` — история версий

## 3) Русские переводы методической зоны

- `ru/SKILL.md`
- `ru/references/*`
- `ru/releases/*`
- `ru/VERSIONS.md`

## 4) Веб-интерфейс GitHub Pages

- `index.html` — UI
- `app.js` — логика (дерево, RU/EN, загрузка, commit)
- `app.css` — стили

## Как использовать репозиторий правильно

1. Для практической работы: редактируйте файлы в блоке "Рабочая зона поиска работы".
2. Для методики и промптов: читайте блок "Interview Coach" и его `ru/` копии.
3. Для быстрых изменений: используйте GitHub Pages интерфейс и кнопку `Commit to GitHub`.
