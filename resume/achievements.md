# Достижения для резюме (оптимизировано под ИИ-скрининг)

## Как использовать

Эти формулировки прошли «нейро-оптимизацию» — содержат ключевые слова, которые ищут рекрутеры и ИИ-скрининг hh.ru/LinkedIn.

**Вставляйте в резюме ТОЧНО в таком виде.**

## 🔑 Ключевые слова для ИИ-скрининга

**Включены в достижения выше:**

| Категория | Ключевые слова |
|-----------|----------------|
| Управление | управление, руководство, команда, отдел, филиалы |
| Продажи | продажи, выручка, маржа, B2B, клиенты, воронка |
| Результат | рост 10-15%, 100%+, ежегодный, в течение 10 лет |
| Цифровизация | CRM, 1C, amoCRM, MS Dynamics, Power BI, дашборды, онлайн-калькуляторы |
| Процессы | бизнес-процессы, регламенты, стандарты, автоматизация, интеграция |
| Кризис | кризис, оптимизация, сокращение, сохранение компетенций |
| Команда | 85 человек, подбор, обучение, мотивация, KPI, наставничество |

## ✅ Чек-лист перед публикацией

- [ ] Все достижения содержат цифры (%, количество, сроки)
- [ ] Ключевые слова из вакансий включены
- [ ] Нет «воды» (ответственный, коммуникабельный)
- [ ] Каждое достижение начинается с глагола действия (открыл, внедрил, обеспечил, разработал)
- [ ] Достижения в резюме совпадают с навыками в профиле
